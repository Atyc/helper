1. Use 'yarn install' to install the dependencies
2. Start webpack using 'yarn run build'
3. Start Live server using 'yarn run serve'
4. Access the server in the browser (usually at http://127.0.0.1:8080/) and observe the console for updates
The script will automatically run with the default data: 
[{ state: 'processing' }, { state: 'error' }]
5. Alter the dummyData array in source.js to test various scenarios
